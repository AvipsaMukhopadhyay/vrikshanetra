🌿 VrikshNetra – AI-Powered Smart Agriculture Assistant

VrikshNetra is an integrated AI-driven agriculture monitoring and decision-support system designed to empower small-scale farmers with data-driven insights. The platform combines multiple intelligent modules to analyze soil, crops, and terrain conditions for better farming decisions.

It includes:

🌍 Terrain Classification – Identifies land type suitability using ML models.

🌾 Crop Recommendation System – Suggests optimal crops based on soil and environmental parameters.

🌱 Soil Health Monitoring – Detects soil moisture and NPK levels using IoT sensors (Arduino/Raspberry Pi integration).

🍃 Crop Disease Detection – Uses deep learning models to identify plant diseases from leaf images.

The system integrates IoT + Machine Learning + Web Dashboard (Flask/Django) to provide real-time monitoring, predictive analysis, and actionable insights for farmers.

Tech Stack: Python, TensorFlow, Scikit-learn, IoT Sensors, Raspberry Pi/Arduino, Flask/Django.
